Aplicativos disponíveis com o i3geo e documentação.

Inicialização normal do i3geo
Atlas  Interface que possibilita a montagem de atlas. Baseia-se no arquivo menutemas/atlas.xml.
Versão zerocal com menos recursos.  Ideal para usuários com capacidade menor de internet.
Versão com o uso do software Flamingo (flash).  Para usuários que gostam de flash.
Versão com o uso do software Openlayers.  Navegação no estilo GoogleMaps baseada no software OpenLayers.
Versão mobile para dispositivos móveis.  Por padrão, a lista de mapas iniciais é montada conforme definido no arquivo menutemas/mapas.xml.
Aplicativo para download de dados.  Por padrão, baseia-se na lista de temas existentes em menutemas/menutemas.xml.
Aplicativo para consulta a web services nos padrões OGC.  Por padrão, A lista de wms é obtida do arquivo menutemas/servicoswms.xml.
Gerador de links  Aplicativo que possibilita ao usuário criar seus próprios links de abertura de mapas customizados.
Testa mapfile  Aplicativo para testar os mapfiles criados para o i3geo.
Testa instalação do i3geo  Aplicativo para testar a instalação do i3geo.
Gerador de endereços de web services.  Aplicativo para listar os endereços que devem ser utilizados para acessar o gerados automático de web services do i3geo (ogc.php).
Gerador web services.  Aplicativo para geração automática de web services OGC com base no menu de temas do i3geo.
Gerador kml.  Aplicativo para geração automática de arquivos kml para uso no Google Earth. Deve ser utilizado dentro do Google Earth na opção "link de rede"
Documentação dos códigos.  Documentação gerada com Naturaldocs.
