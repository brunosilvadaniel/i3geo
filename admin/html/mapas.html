Administração do i3geo - Links para mapas

Ajuda

Vc pode registrar qualquer link para mapas. A opção "editar" abre um formulário para a criação do link conforme os parâmetros utilizados pelo i3Geo. Você pode desprezar todos os campos do formulário e incluir apenas o link direto desejado. Pode ser armazenado um mapfile completo com todas as configurações do mapa
