Metadados estatísticos - editor de tabelas de dados

Ajuda

Este formulário permite editar as tabelas que contém os dados utilizadas pelo sistema de metadados estatísticos

Atenção: para poder editar o banco de dados, o usuário da conexão escolhida deve ter direitos de edição. Apenas as tabelas e os dados existentes no esquema i3geo_metaestat podem ser editados
