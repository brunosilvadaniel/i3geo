Metadados estatísticos - upload de dados

Ajuda

Este formulário permite fazero upload de dados utilizadas pelo sistema de metadados estatísticos


Atenção: para poder editar o banco de dados, o usuário da conexão escolhida deve ter direitos de edição. Apenas variáveis existentes no esquema i3geo_metaestat podem receber dados dessa forma.
O arquivo deve ser do tipo texto e a primeira linha deve conter os nomes das colunas
