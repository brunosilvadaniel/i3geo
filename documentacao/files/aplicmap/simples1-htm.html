Variaveis

g_locaplic

Indica a localização correta do i3geo.

É utilizada para identificar o local correto onde estão os programas em php que são utilizados.

g_localimg

Indica a localização correta das imagens.

É utilizada para identificar o local correto onde estão as imagens que são utilizados.

g_funcaoTip

Função ajax que será executada para mostrar informações do tipo TIP.

A função é executada pelo CPAINT e avaliada com “eval”.

Default

g_funcaoTip = “verificaTipDefault()”;

g_tempotip (opcional)

Tempo utilizado para verificar se o mouse está parado.

Se o mouse estiver parado, a função de mostrar tip é ativada.

Default

g_tempotip = 1000;

g_tipotip

Define como o tip será mostrado.

Pode ser definida como “simples” ou “completo”

Default

g_tipotip = “completo”;

g_tipoimagem

Indica o tipo de filtro de imagem que está ativo.  O filtro ativo é aplicado sobre a imagem toda a vez que o mapa é refeito.

Default

g_tipoimagem = “nenhum”;

destacaTamanho

Valor em pixel do retângulo de destaque de temas.

Default

destacaTamanho = 75;

g_entorno

Indica se o preenchimento do entorno do mapa está ou não ativo.

Utilizado para criar o efeito de auto-preenchimento do mapa quando é executada a função pan.

É alterada em uma opção específica no menu suspenso.

Default

g_entorno = “nao”;

g_guiaativa

Indica qual guia do mapa iniciará ativa.

Default

g_guiaativa = “guia1”;

g_diminuixM

Diminui a largura do mapa em pixels no caso do navegador ser o IE.

Default

g_diminuixM = 20;

g_diminuixN

Diminui a largura do mapa em pixels no caso do navegador ser o FF.

Default

g_diminuixN = 20;

g_diminuiyM

Diminui a altura do mapa em pixels no caso do navegador ser o IE.

Default

g_diminuiyM = 150;

g_diminuiyN

Diminui a altura do mapa em pixels no caso do navegador ser o FF.

Default

g_diminuiyN = 120;

g_janelazoom

Define se a barra de zoom será aberta no mapa.

Default

g_janelazoom = “sim”

g_barraFerramentas1

Define se a barra de ferramentas 1 será mostrada

Defaul

g_barraFerramentas1 = “sim”

g_barraFerramentas2

Define se a barra de ferramentas 2 será mostrada

Defaul

g_barraFerramentas2 = “sim”

g_mapaRefDisplay

Indica a visibilidade do mapa de referência na inicialização

Pode ser “block” ou “none”.

g_docaguias

Variável que define se o mapa deve iniciar com as guias em janela ou não.

Default

g_docaguias = “nao”;

g_downloadbase

Variável que define se na guia 2 será mostrada a opção de download.

Default

g_downloadbase = “sim”;

g_conectargeorss

Variável que define se na guia 2 será mostrada a opção de conexão com GeoRSS.

Default

g_conectargeorss = “sim”;

g_uploadlocal

Variável que define se na guia 2 será mostrada a opção de upload.

Default

g_uploadlocal = “sim”;

g_conectarwms

Variável que define se na guia 2 será mostrada a opção de conexão com WMS.

Default

g_conectarwms = “sim”;

g_janelaMen

Variável que define se a janela de mensagens começará aberta.

Default

g_janelaMen = “sim”;

g_mostraRosa

Variável que define se a rosa dos ventos deve ser mostrada junto ao mouse

Default

g_mostraRosa = “sim”

g_opcoesTemas

Variável que define se as opcoes adicionais de cada tema serao mostradas.

Default

g_opcoesTemas = “sim”;

g_3dmap

Variável que define o nome do map_file que possuí o layer para uso na função 3d.

Pode ser caminho completo.  Se não, busca no aplicmap.

g_embedLegenda

Indica se a legenda deve ser incluida no corpo do mapa.

Default

g_embedLegenda = “nao”;

g_mashuppar

Parâmetros de inicialização que podem ser utilizados na interface mashup.

Os parâmetros são os mesmos que podem ser utilizados quando o i3geo é inicializado pelo ms_criamapa.php.

Exemplo: g_mashuppar = “&pontos=-54 -12&temasa=biomas&layers=biomas”

g_mashuppar = “&mapext=-53 -26 -42 -18”;

g_mensagempadrao

Mensagem padrão que será mostrada na janela de mensagens.

g_mensagempadrao = “O I3Geo é software livre!  Para download clique <a href=’url target=”http://mapas.mma.gov.br/download” name=”http://mapas.mma.gov.br/download”’ target=blanck >aqui</a>”;

g_visual

Indica qual o tipo de visual para abertura do mapa.

Os visuais disponíveis são obtidos do diretório i3geo/imagens/visual na inicialização do i3geo.

g_autoRedesenho

Ativa o auto redesenho após um determinado tempo (milisegundos).

Após decorrido o tempo definido, o mapa é redesenhado.  Se for 0 o temporizador não é ativado.

Define qual div corresponde ao local de listagem das camadas disponíveis no mapa

objmapa. guiaMenu

Define qual div corresponde ao local de inclusão do menu de temas

objmapa. guiaLegenda

Define qual div corresponde ao local onde será incluida a legenda em HTML
