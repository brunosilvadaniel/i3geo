Para a montagem de cada gráfico é necessário selecionar um tema que contenha os dados que serão representados.

Os dados são aqueles que encontram-se na tabela de atributos do tema.

Após selecionado o tema, é mostrada uma lista com os itens existentes na tabela de atributos. Escolha os itens que serão utilizados no gráfico clicando no box que aparece na lista de itens. Cada item irá compor uma fatia do gráfico. Ao lado do item, é mostrada a cor que será utilizada na representação. Cada cor é definida em RGB.

Na guia "propriedades" é possível definir propriedades de representação de cada gráfico. As definições afetarão a forma de cada gráfico.

Cada gráfico adicionado &etilde; incluído no mapa como um novo tema.
