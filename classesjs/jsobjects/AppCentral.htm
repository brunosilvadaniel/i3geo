jsObjects AppCentral - Version 1.0 By: Jonathan Wise 

About

  • AppCentral is a web application container based on jsObjects and a .NET webservice, that combines modules -- both client and server-side, to assemble disperate web applications into a cohesive interface.

    - Change Log

Purpose

  • The web is becoming much more than a place to share data. Technologies like web services and AJAX are allowing developers to build rich, interactive applications that rival their compiled equivalents on the desktop. Unfortunately, differing development styles, interfaces and appearances make switching between applications an experience no better than it was 5 years ago. AppCentral is an attempt to provide a single interface through which multiple, disperate applications can be accessed in a user-friendly and consistent manor.
    AppCentral can host "modules" or web applications that adhere to a few standards for consistency of appearance and data retreival, or it can host standard web pages, requiring only a small XML file and icon to allow them to appear in the container.

Requirements

  • AppCentral is intended for use in both Internet Explorer (5.5 or better) and FireFox (1.1 or better) and uses a .NET webservice on the server-side.
